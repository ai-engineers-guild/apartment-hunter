# Apartment Hunter

Apartment Hunter — это агрегатор и платформа для поиска аренды квартир. Инструмент автоматизирует поиск, фильтрацию и скоринг жилья, избавляя от необходимости вручную обновлять сайты. **В данный момент система поддерживает работу только с `krisha.kz`**, но архитектура позволяет легко добавлять новые источники. Поддерживает точный поиск по гео-полигонам, семантический текстовый поиск (RAG) и оценку качества с помощью нейросетей (LLM).

## Быстрый старт

```powershell
# 1. Запуск MCP-сервера для работы с Codex Desktop / Claude
uv run apartment-hunter-mcp

# 2. Или ручной запуск пайплайна сбора новых объявлений
uv run apartment-hunter-ingest
```

### AI-Навыки (Skills)
В проекте есть готовые навыки для вашего AI-ассистента (в `.agents/skills`):
- **apartment-search**: Поиск квартир, мониторинг и перевод риелторского жаргона.
- **city-district-context-kz**: Контекст по городам и районам (транспорт, цены, инфраструктура).
- **html-report-generator**: Генерация визуального HTML-отчета с фото и параметрами квартир и запуск его на локалхосте.

**Как установить навыки:**
Вам не нужно устанавливать эти навыки вручную. Просто откройте папку этого проекта через ваш AI CLI (например, Antigravity CLI / Codex CLI), и агент автоматически обнаружит и загрузит все локальные навыки из рабочего пространства.

## Примеры использования (Use Cases)

- **AI-ассистенты (MCP):** Подключите к Codex Desktop и общайтесь с вашей базой жилья естественным языком ("Покажи двушки возле Мегапарка до 500к, отсортируй по новизне").
- **Точная география:** Забудьте про неточный текстовый поиск по названиям улиц. Нарисуйте полигон на карте, и система найдет квартиры строго внутри заданных границ.
- **Умный скоринг:** Автоматическая оценка квартир (от 1 до 10) на основе качества ремонта, состояния мебели и ваших личных предпочтений с помощью Vision LLM.
- **Мониторинг рынка:** Запускайте сбор данных по расписанию (через cron/task scheduler), чтобы отслеживать падение цен и мгновенно ловить свежие варианты до того, как их снимут.

## Архитектурная философия

Это **Library-First** проект. Вся базовая логика реализована как переиспользуемая Python-библиотека. Взаимодействие с конкретными сайтами вынесено в изолированные адаптеры-плагины, что делает ядро стабильным и легко расширяемым.
